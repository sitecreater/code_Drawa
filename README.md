## 소개
**Drawa 웹 사이트 프로젝트 개발에 사용 된 코드 입니다.**

## Drawa 사이트란?
**한정판 신발과 옷을 구매 혹은 응모 할 수 있는 사이트이며 그 밖에 시세확인, 장바구니, 모바일 navbar 기능이 있습니다.**

## 포트폴리오 페이지에 사용한 기술
### React, CSS, Firebase, Firebase Hosting

## 반응형 웹페이지
PC, tablet, mobile 해상도에 따라 화면이 맞춰집니다.
